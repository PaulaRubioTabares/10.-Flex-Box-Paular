Flex-Box

Modelo de diseño en HTML "cajas flexibles"

Lo primero a tener en cuenta es que los elementos a los que se les quiere modificar distribución con este método deben estar dentro de un contenedor, al cual se le aplica la propiedad de display flex o inline-flex
